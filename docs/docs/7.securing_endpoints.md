# Securing Endpoints

Endpoints are secured using:


@SecuredEndpoint


---

# Example

```java
@GET
@Path("/books/{bookId}")
@SecuredEndpoint(
   params = {
       @ParamRef(
           param = "bookId",
           type = ParamType.PATH,
           referTo = BookResource.class
       )
   }
)
public Response getBook(@PathParam("bookId") String bookId) {
   return Response.ok().build();
}

```

What Happens?

Request:

```http
GET /books/BOOK-1
```
**Step 1 — Role Check**<br/>
User must have endpoint role (e.g. BOOK-ADMIN)<br/>

**Step 2 — Extract Resource**<br/>
bookId = BOOK-1<br/>
resource = BOOK <br/>
**Step 3 — Build Entitlement**<br/>
BOOK-ADMIN / BOOK / BOOK-1 <br/>
**Step 4 — Authorization Check**<br/>
If match exists → Access Granted
Else → Access Denied
